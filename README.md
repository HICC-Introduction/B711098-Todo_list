## 프로젝트 설명
일정을 추가 및 편집해 관리할 수 있도록 만든 웹페이지입니다.

웹페이지에서 '할 일'들을 추가, 수정, 삭제, 완료처리, 우선순위 설정할 수 있습니다.



## 구현한 기능
* 할 일의 추가, 수정, 삭제, 완료 처리
* 할 일의 우선순위를 설정 및 조절



## 사용한 기술 & 언어
* CSS
* HTML
* JavaScript
* JQuery
